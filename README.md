# Compare and Buy Insurance Prototype

This is a mobile-first HTML prototype . The page simulates an embedded car insurance results experience inside a car buying app, with quick filters, plan cards, plan details, and a compare flow.

## What’s included

* Quick filters for common coverage needs
* Horizontally scrollable insurance plan cards
* Plan detail view with coverage, exclusions, and add-ons
* Compare mode for selecting multiple plans
* Side-by-side comparison view
* Select Plan confirmation state
* Empty state when no plans match the selected filters

## Notes

* The data in this prototype is fictional.
* The page is designed to work without a build step.
* It is suitable for GitHub Pages hosting.
